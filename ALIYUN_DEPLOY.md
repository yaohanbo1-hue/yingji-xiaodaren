# 阿里云函数计算 FC 部署指南

## 部署步骤（约 10 分钟）

### 1. 注册阿里云账号

1. 访问 https://www.aliyun.com
2. 点击 "免费注册"，用**手机号**注册
3. 完成实名认证（个人认证即可，免费）

### 2. 开通函数计算服务

1. 登录阿里云控制台：https://www.aliyun.com/product/fc
2. 点击 "立即开通"（免费开通，不需要付费）
3. 选择 "函数计算 FC 3.0"

### 3. 创建函数

1. 进入函数计算控制台
2. 选择地域（推荐：**华东 1 杭州** 或 **华东 2 上海**）
3. 点击 "创建函数"
4. 选择创建方式：
   - **创建方式**：使用标准 Runtime 创建
   - **函数名称**：`yingji-ai-proxy`
   - **运行环境**：Python 3.10
   - **函数入口**：`aliyun-fc.handler`（默认格式：文件名.函数名）
5. 代码部署方式：
   - 选择 "使用代码 ZIP 包部署"
   - 或者直接在控制台粘贴代码
6. 点击 "创建"

### 4. 上传代码

如果控制台直接粘贴：
1. 进入函数详情 → 代码页签
2. 选择 "代码编辑器" 模式
3. 删除默认代码，把 `aliyun-fc.py` 的内容全部粘贴进去
4. 点击 "部署代码"

如果上传 ZIP 包：
1. 把 `aliyun-fc.py` 打包成 ZIP
2. 函数详情 → 代码页签 → 上传 ZIP 包
3. 点击 "部署代码"

### 5. 设置环境变量

1. 函数详情 → 配置页签 → 环境变量
2. 添加变量：
   - **变量名**：`DEEPSEEK_API_KEY`
   - **变量值**：`sk-4cdcfcde0e1343789c07266c23efd371`
3. 点击 "保存"

### 6. 配置 HTTP 触发器

1. 函数详情 → 触发器页签 → 创建触发器
2. 触发器类型：**HTTP 触发器**
3. 触发器名称：`http-trigger`
4. 认证方式：**不需要认证**（或者选择 "Header 认证" 更安全）
5. 请求方式：勾选 **POST** 和 **OPTIONS**
6. 点击 "创建"
7. 创建后会获得一个 URL，类似：
   ```
   https://yingji-ai-proxy-abc123.defgh.aliyuncs.com
   ```

### 7. 测试函数

1. 在触发器页面找到 HTTP 地址
2. 用浏览器或 curl 测试：
   ```bash
   curl -X POST "你的HTTP触发器URL" \
     -H "Content-Type: application/json" \
     -d '{"message":"地震来了怎么办？"}'
   ```
3. 如果返回 JSON 包含 `answer` 字段，说明部署成功！

### 8. 配置前端

打开你的 GitHub Pages 页面，按 F12 控制台执行：

```javascript
localStorage.setItem('deepseek_proxy_url', 'https://你的HTTP触发器URL');
```

刷新页面，AI 就会通过阿里云调用真正的 DeepSeek！

## 常见问题

**Q: 阿里云需要付费吗？**
A: 函数计算有免费额度，个人使用完全够用：
- 每月 100 万次调用
- 每月 40 万 GB-秒 计算资源
- 你的小应用一个月通常不会超过 1000 次调用

**Q: 如何查看用量？**
A: 函数计算控制台 → 监控指标 → 调用次数

**Q: 触发器 URL 可以自定义域名吗？**
A: 可以。在触发器页面绑定你自己的域名（需要域名在阿里云备案或解析）。

**Q: 代码粘贴进去后报错？**
A: 检查函数入口是否设置为 `aliyun-fc.handler`（文件名和函数名之间用点号分隔）。

**Q: 调用返回 500 错误？**
A: 检查环境变量 `DEEPSEEK_API_KEY` 是否正确设置。

## 安全提醒

⚠️ **不要把 API Key 写在代码里！** 一定要通过阿里云的环境变量设置，这样更安全。
