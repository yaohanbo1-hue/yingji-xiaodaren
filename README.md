# 🌪️ 应急小达人 v1.2.0

> **Disaster Blind Box Command HQ**
> 
> 一款面向中小学生的防灾教育互动游戏，通过盲盒开箱、答题闯关、情景模拟等趣味方式学习防灾知识。

## ✨ 项目亮点

### 🎮 28 种游戏模式
| 分类 | 模式数量 | 代表功能 |
|------|----------|----------|
| 📚 学习中心 | 10个 | 开盲盒、签到、学习、急救、防灾馆、故事书库、卡牌工坊、百科、引导、AI导师 |
| 🏅 认证与模拟 | 3个 | 能力认证、灾害模拟、真实案例 |
| ⚔️ 闯关挑战 | 18个 | 大擂台、速答、双人PK、生存、故事冒险、精准打击、记忆翻牌等 |

### 🌊 四大核心模块

| 模块 | 功能 | 教育价值 |
|------|------|----------|
| 🤖 **AI 智能导师** | 12维雷达图 + 智能诊断 + 对话终端 + 推荐练习 | 个性化学习、数据驱动 |
| 🏅 **能力认证** | 5级认证 + Canvas证书 + 进度环 + 升级弹窗 | 激励机制、成就体系 |
| 🎬 **灾害模拟** | 4种灾害Canvas动画 + 真实决策 + 粒子系统 | 沉浸体验、情景教学 |
| 📖 **真实案例** | 5个历史事件 + 时间线 + 三阶段流程 + 知识总结 | 科学性、教育价值 |

### 📊 学习数据系统
- **📕 错题本**：自动收集错题，支持专项复习
- **📊 学习报告**：一键生成PNG报告，可打印提交
- **🔊 语音播报**：Web Speech API 朗读知识点
- **📈 进度追踪**：12类灾害掌握情况可视化

## 🚀 快速开始

### 使用方法

```
1. 下载 ZIP 压缩包
2. 解压到任意目录
3. 双击 index.html 即可运行
```

**无需安装任何软件！无需联网！无需服务器！**

### 系统要求
- Chrome 80+ / Edge 80+ / Firefox 75+ / Safari 13+
- 支持移动端（iOS 13+ / Android 10+）
