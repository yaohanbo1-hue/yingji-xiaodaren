🧹 应急小达人 — 缓存清除工具

如果你看到的内容还是旧版本，按以下步骤操作：

方法 1：清除 Service Worker（推荐）

按 F12 打开控制台，粘贴以下代码执行：

// 清除所有缓存和 Service Worker navigator.serviceWorker.getRegistrations().then(regs => { regs.forEach(r => r.unregister()); console.log('✅ Service Worker 已清除'); }); caches.keys().then(names => { names.forEach(name => caches.delete(name)); console.log('✅ 缓存已清除'); }); localStorage.clear(); console.log('✅ 本地存储已清除'); setTimeout(() => location.reload(true), 1000);
